# Race Across The World Game
A strategy game inspired by the BBC TV show Race Across The World

## Concept
- A single-player game with computer opponents
- Goal is to travel from origin city to destination city and arrive before your opponents
- All players start with the same budget, equivalent to the price of an economy airfare from origin to destination
- You can only travel over land and sea (no flights allowed)
- You can choose one of several routes from the city you are currently at in order to move closer to the destination
- Each route option has an associated mode of transport, time taken, cost, distance from current city and distance to destination
- Players have to balance speed against budget in order to reach the destination before their opponents
- Game design should remain as simple as possible for the first version
